Необходимо вывести описание структуры в табличном виде со следующими столбцами:
"Наименование элемента/атрибута", "Тип данных", "Обязательность", "Описание", "Фрагмент xml-файла"
При формировании таблицы требуется обеспечить следующие требования к оформлению:

1. Выделять корневой элемент в отдельной строке. В столбце описание для корневого элемента писать "Корневой элемент". В
   столбцах "Тип данных" и "Обязательность" ставить прочерки.
2. Выделять группирующий элемент, включающий блок элементов. В столбце описание для корневого элемента писать "
   Группирующий элемент". В столбцах "Тип данных" и "Обязательность" ставить прочерки.
3. Для дочерних элементов заполняются все столбцы. В описании указывать "Дочерний элемент элемента "{{Имя группирующего
   элемента}}""

```
− Ставить «Да», если поле обязательно для заполнения!!!!

− Ставить «−» для интерактивных элементов.
```
